TypeMeNot

A Chrome extension that checks for toxic characteristics in the
text that you are composing for social media sites, in real time.

Try it yourself:

  1. Download the extension here
  2. Navigate to chrome://extensions/
  3. Enable developer mode by using the toggle in the top right
  4. Click Load unpacked and select the extension
